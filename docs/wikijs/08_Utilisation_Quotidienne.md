# 08 - Utilisation quotidienne

## Objectif

Guider les utilisateurs dans l'usage courant du wiki: creation, edition, recherche et gouvernance du contenu.

## Profils utilisateurs

- Lecteur: consulte et recherche l'information
- Editeur: cree et met a jour les pages
- Admin: gere droits, structure et parametres

## Parcours lecteur

1. Ouvrir l'accueil du wiki.
2. Utiliser la recherche globale avec mots-cles.
3. Naviguer via les categories et tags.
4. Verifier la date de derniere mise a jour de la page.

## Parcours editeur

1. Creer une page depuis le bon espace.
2. Utiliser le template de page standard.
3. Renseigner titre, contexte, procedure, validation.
4. Ajouter tags et liens vers pages associees.
5. Publier puis annoncer la mise a jour si necessaire.

## Standard de redaction

- Un seul sujet par page
- Titres clairs et actionnables
- Etapes numerotees pour les procedures
- Captures ecran seulement si utiles
- Ajouter une section "Derniere mise a jour"

## Bonnes pratiques d'utilisation

- Rechercher avant de creer une nouvelle page
- Mettre a jour l'existant plutot que dupliquer
- Utiliser des tags constants
- Eviter les acronymes sans definition
- Valider les pages critiques a 2 personnes

## Regles de qualite

- Aucune page "orpheline"
- Chaque page critique a un proprietaire
- Revue documentaire mensuelle
- Archivage des pages obsoletees
